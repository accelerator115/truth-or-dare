# 真心话大冒险

![版本](https://img.shields.io/badge/version-1.0.0-blue.svg)
![uni-app](https://img.shields.io/badge/uni--app-Vue3-green.svg)
![平台支持](https://img.shields.io/badge/平台-微信小程序|H5|App-orange.svg)

一个基于 uni-app 框架开发的真心话大冒险小游戏，支持多平台部署（微信小程序、H5、Android/iOS App等）。

##  项目简介

这是一个专为聚会和朋友聚餐设计的互动游戏应用，玩家可以通过转动精美的轮盘随机选择真心话问题或大冒险任务。应用提供了完善的内容管理、新手指引、历史记录等功能，让每次聚会都充满乐趣和惊喜。

##  项目特色

-  **现代化UI设计**：采用卡片式布局和渐变色彩，界面美观大方
-  **流畅转盘动画**：精心设计的转盘旋转效果，增强游戏体验
-  **多平台兼容**：一套代码，支持微信小程序、H5、Android/iOS App
-  **本地数据存储**：无需网络连接，所有数据本地保存
-  **Vue3 Composition API**：采用最新的Vue3语法，代码结构清晰
-  **完善的新手指引**：首次使用自动引导，降低学习成本

##  主要功能

###  游戏模式
- **真心话模式**：回答各种有趣的个人问题
- **大冒险模式**：完成各种有趣的挑战任务
- **智能模式切换**：一键切换游戏模式，自动保存每个模式的上次选择结果

###  新手指引系统
- **首次启动指引**：新用户首次打开应用时自动显示操作指南
- **交互式教程**：逐步引导用户了解各个功能区域和操作方法
- **随时查看**：点击右上角"?"按钮可重新查看操作指引
- **智能推荐**：完成指引后智能推荐导入默认内容包

###  内容管理系统
- **自定义添加**：手动添加真心话问题或大冒险任务
- **重复检测**：智能检测并阻止重复内容的添加
- **批量导入**：一键导入预设的默认内容包（每个模式10个精选选项）
- **分类显示**：清晰区分真心话和大冒险内容，支持按类型筛选
- **删除管理**：支持单个删除不需要的内容

###  转盘游戏功能
- **公平随机算法**：采用真正的随机算法确保每次选择的公平性
- **流畅动画效果**：精心设计的转盘旋转动画，视觉体验佳
- **手动控制**：支持手动停止转盘，增加游戏互动性
- **最少限制**：每个模式至少需要3个选项才能开始游戏
- **结果展示**：清晰显示选中结果和对应的游戏模式

###  历史记录管理
- **完整记录**：保存所有游戏历史，包括内容、类型和执行时间
- **智能限制**：最多保留20条历史记录，自动清理最旧记录
- **时间排序**：按时间倒序显示，最新记录在前
- **类型标识**：清楚标识每条记录是真心话还是大冒险

###  数据管理功能
- **本地存储**：所有数据保存在本地，无需网络连接，保护隐私
- **多级清理选项**：
  - 清空历史记录（保留所有选项内容）
  - 清除所有选项（保留历史记录）
  - 重置所有数据（完全重置到初始状态）

##  快速开始

###  使用方法

#### 1️⃣ 新手指引（首次使用）
- 首次打开应用时，系统会自动显示交互式新手指引
- 按照指引逐步了解各个功能的使用方法
- 可以点击"跳过引导"快速进入游戏
- 完成指引后，点击右上角的"?"按钮可重新查看

#### 2️⃣ 选择游戏模式
- 在顶部模式选择器中点击"真心话"或"大冒险"按钮
- 切换模式时会自动显示该模式的上次选择结果
- 不同模式的内容会自动过滤显示

#### 3️⃣ 添加游戏内容
**方法一：手动添加**
1. 在输入框中输入真心话问题或大冒险任务
2. 点击"添加"按钮或按回车键确认
3. 内容会自动分类到当前选择的模式下
4. 系统会自动检测重复内容

**方法二：导入默认内容**
1. 点击"导入默认内容"按钮
2. 系统会自动导入10个精心挑选的预设内容
3. 不会重复导入已存在的内容

#### 4️⃣ 开始游戏
1. 确保当前模式下至少有3个选项
2. 点击"开始旋转"按钮启动轮盘
3. 等待转盘自动停止，或点击"停止旋转"手动停止
4. 查看选中的结果并执行相应任务

#### 5️⃣ 查看历史记录
- 在页面底部可以查看所有游戏历史
- 历史记录按时间倒序排列，最新在前
- 显示任务类型、内容和执行时间
- 最多保留20条记录

#### 6️⃣ 数据管理
- **清除选项**：删除当前所有选项，保留历史记录
- **清空历史记录**：删除所有历史记录，保留选项
- **重置所有数据**：完全重置应用到初始状态

##  开发部署

### 环境要求
- **Node.js**: 14.0+ 
- **开发工具**: HBuilderX 或 VS Code + uni-app插件
- **调试工具**: 微信开发者工具（用于小程序调试）
- **Git**: 版本控制工具

###  开发步骤

1. **克隆项目**
```bash
git clone https://github.com/accelerator115/truth-or-dare.git
cd truth-or-dare
```

2. **安装依赖**
```bash
npm install
```

3. **开发调试**
   - 使用 HBuilderX 打开项目
   - 选择运行平台进行调试：
     - 微信小程序：需要微信开发者工具
     - H5：直接在浏览器中运行
     - App：需要相应的模拟器或真机

4. **项目配置**
   - 修改 `manifest.json` 中的应用信息
   - 在 `pages.json` 中配置页面路由和样式
   - 根据需要调整全局样式 `uni.scss`

5. **构建发布**
   - 微信小程序：发行 → 小程序-微信
   - H5：发行 → 网站-PC Web或手机H5
   - App：发行 → 原生App-云打包

###  平台特定配置

#### 微信小程序
- 需要在微信公众平台注册小程序
- 配置 `manifest.json` 中的微信小程序 appid
- 注意微信小程序的限制和规范

#### H5 网页
- 可直接部署到任何Web服务器
- 支持响应式设计，适配移动端和桌面端

#### Android/iOS App
- 使用 uni-app 的云打包服务
- 可配置应用图标、启动页等
- 支持原生插件扩展

##  自定义配置

### 修改默认内容
在 `pages/index/index.vue` 中找到以下数组，可以修改预设的游戏内容：

```javascript
// 默认真心话内容
const defaultTruth = [
  "你最尴尬的一次经历是什么？",
  "你有什么不为人知的特殊技能？",
  // ... 更多内容
];

// 默认大冒险内容  
const defaultDare = [
  "学动物叫30秒",
  "给任意一个朋友发送一条搞笑信息",
  // ... 更多内容
];
```

### 自定义新手指引
在 `components/guide/guide.vue` 中的 `guideSteps` 数组可以自定义指引步骤：

```javascript
const guideSteps = [
  {
    title: "选择游戏模式",
    content: "点击这里选择真心话或大冒险模式",
    target: ".mode-selector",
    // ... 其他配置
  },
  // ... 更多步骤
];
```

### 主题色彩自定义
1. **导航栏颜色**：在 `pages.json` 中修改 `navigationBarBackgroundColor`
2. **全局样式**：在 `uni.scss` 中定义全局CSS变量
3. **组件样式**：在各组件的 `<style>` 部分修改具体颜色

### 游戏规则调整
- **最少选项数量**：修改 `filteredItems.length >= 3` 中的数字
- **历史记录数量**：修改 `history.value.length > 20` 中的数字
- **转盘速度**：在 `wheel.vue` 组件中调整动画持续时间

### 响应式断点
```css
/* 小屏设备 */
@media (max-width: 480px) { }

/* 中等屏幕 */
@media (min-width: 481px) and (max-width: 768px) { }

/* 大屏设备 */
@media (min-width: 769px) { }
```
